Graph pull requests from any Github repository


How to use this bookmarklet

  1. Drag this link, Graph PRs, to your bookmarks bar.
  2. Visit a Github repository for which you’d like to graph the open pull requests over time.
  3. Click the bookmarklet and a graph will open in a new window (once popups have been allowed, if necessary).

Can’t find your bookmarks bar?

Try these instructions to enable your bookmarks bar: Chrome, Firefox, Safari.
